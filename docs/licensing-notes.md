# Licensing Notes

Magent uses a split license:

- repository code and tooling: MIT
- original research content, catalog metadata, reviews, and documentation:
  CC BY 4.0
- imported source material: keeps its original terms; avoid copying and cite
  instead

Do not copy large problem statements, survey text, or encyclopedia text into this repository unless the license and attribution requirements are understood.

The safest default is to store original concise summaries plus precise source links.

See the root license files and `THIRD_PARTY_NOTICES.md` for the authoritative
project licensing story.
