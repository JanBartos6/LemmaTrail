# Contributors

LemmaTrail recognizes useful research-state work, not only code commits.

Contributions may include:

- proposing a checkable claim, route, failure, or next task
- finding or correcting a source
- reviewing a derivation for gaps
- formalizing an idea into a template
- curating an issue, comment, or model output into durable repository state

Git history records commits. Research objects should also preserve visible
credit in frontmatter when practical.

AI tools must be disclosed in `ai_assistance`, but they are tools, not authors.
Credit goes to the humans who contribute, review, curate, or submit the work.
