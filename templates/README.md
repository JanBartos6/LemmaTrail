# Research Object Templates

These templates are for individual research objects inside an active problem
workspace.

Use them under paths such as:

```text
problems/<problem-id>/proposals/claims/
problems/<problem-id>/proposals/routes/
problems/<problem-id>/proposals/attempts/
problems/<problem-id>/refuted/
problems/<problem-id>/review/
```

Do not use this directory to start a new problem workspace. For that, copy
`problems/_workspace-template/`.

Keep every object small, checkable, source-backed or derivation-backed, and
explicit about its gap or next continuation point when applicable.
